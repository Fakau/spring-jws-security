## To get the token, you should make a post request
## Link:
localhost:8080/api/authenticate
## Request Body
{
    "login" : "admin",
    "password" : "admin"
}
See the swagger for more


## The link to see the swagger-ui 
http://localhost:8080/swagger-ui.html#/
